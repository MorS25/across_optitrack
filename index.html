Welcome to ACROSS Optitrack GitHub Pages.

This project aims to explain, in detail, how to use Optical Motion Capture Systems and Tracking Software (Optitrack) in a Laboratory network setup with multiple PC clients. The project covers the basics from setting up the Motion Tracking Hardware and Software, to the actual implementation of a ROS (Robot Operating System) based client application.
